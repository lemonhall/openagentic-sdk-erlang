Performs exact string replacements in a file within the workflow/session workspace (NOT the project repo).

Workspace root (write allowed):
- {{workspace_dir}}

Project root (read-only via other tools):
- {{project_dir}}

Usage:
- Provide `file_path` (or `filePath`), `old` (or `old_string` / `oldString`), and `new` (or `new_string` / `newString`).
- Paths are resolved under the workspace root; any path outside the workspace root will be rejected.
- Use `replace_all=true` (or `count=0`) to replace all occurrences. Otherwise, a single occurrence is replaced by default.
- If `old` is not found, the tool fails.
