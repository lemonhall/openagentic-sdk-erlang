Creates or overwrites a file within the workflow/session workspace (NOT the project repo).

Workspace root (write allowed):
- {{workspace_dir}}

Project root (read-only via other tools):
- {{project_dir}}

Usage:
- Provide `file_path` (or `filePath`) and `content`.
- Paths are resolved under the workspace root; any path outside the workspace root will be rejected.
- If overwriting an existing file is intended, set `overwrite=true`.
- Prefer `Edit` when making small, precise changes to an existing file.
