Finds files by glob pattern within the project repo or the workflow/session workspace.

Project root:
- {{project_dir}}

Workspace root:
- {{workspace_dir}}

Usage:
- Provide `pattern` (e.g., `**/*.md`, `skills/**/SKILL.md`).
- Optional: restrict the search base with `root` (or `path`). Relative paths are resolved against the project root.
- Any path outside the project root will be rejected.
- To search under the workspace, set `root` to a `workspace:` path (e.g. `workspace:.` or `workspace:deliverables`).
